Final Year Project

World-Seed Generation Tool

For my final year project at university I selected Design-Time Procedural generation as my focus. Instead of focusing on how PCG (Procedural Content Generation) can be used at run-time, This project explores how it can instead be used by designers to speed up their blockout and itterative phases.

For this project, the implementation utilises UE4 and acts as a plugin that allows the designers to quickly generate dungeon style enviroments by utilising "Landmarks" placed throughout the scene.
